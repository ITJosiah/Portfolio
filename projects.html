Web Development

Christian Foundational Learning Center (CFLC) Official Website

A comprehensive online platform designed for Christian Foundational Learning Center, a private, non-profit school located in Basud, Camarines Norte. The primary goal of the website is to provide a digital presence that informs parents and the community about the school's academic offerings and spiritual foundation.

Key Features

  • Engaging Homepage — Showcases the school's motto, "Bawat bata sa Diyos ay mahalaga," and the current school year theme, "UNITY in VISION".
  • School Profile — Dedicated "About Us" section detailing the school's history, core values (LEGACY), and its mission and vision.
  • Admission Portal — Clear enrollment guide for Pre-School pupils, Grade 1 pupils, and transferees.
  • Interactive Resources — Quick-access buttons for curriculum, handbook, calendar, and uniform guides.
  • Information Gallery — High-quality images showcasing the campus, classrooms, and student activities.
  • Contact & Location — Direct messaging form, school contact details, and an integrated Google Maps embed.

Technologies Used

HTML5 CSS3 JavaScript Git & GitHub GitHub Pages Google Maps API

Desktop Application

CFLC Student Information Management System

A centralized, database-driven desktop application developed to digitalize the administrative operations of Christian Foundational Learning Center (CFLC). The system replaces manual paper-based record-keeping with a secure platform to manage the complete lifecycle of school data.

Key Modules & Functionalities

  • Role-Based Access Control — Three user levels: Super Admin (system config & accounts), Administrator (core data), and Teacher (viewing & reports).
  • Comprehensive Data Management — Modules for Students, Teachers, Sections, Subjects, and Enrollments with full CRUD operations.
  • Enrollment Tracking — Tracks school year placements, payment statuses (Paid, Partial, Unpaid), and academic year summaries.
  • Advanced Reporting — Automated, printable reports including Master Lists, Enrollment Summaries, Teacher Loads, and Section Analytics.
  • System Integrity & Security — Local database backups, maintenance mode, and secure credential-based authentication.

Technologies Used

VB.NET Visual Studio 2022 MySQL XAMPP Entity Framework Figma Lucid Chart Windows 11
